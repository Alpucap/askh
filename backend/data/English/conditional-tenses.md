# Conditional Tense
## Conditional Sentence Type 0

> **Simply:** General Truth / Cause and Effect.

**Formula:** If + Simple Present, Simple Present

**Usage:** Commonly used for general truths, scientific facts, or habits where the result is certain.

**Examples:**
(+) If you heat butter, it melts.
(-) If you do not heat butter, it does not melt.
(?) Does butter melt if you heat it?

## Conditional Sentence Type 1

> **Simply:** Real Possibility in Future.

**Formula:** If + Simple Present, Simple Future (will + V1)

**Usage:** Commonly used for a real possibility in the future; things that are likely to happen if a condition is met.

**Examples:**
(+) If I pass the exam, I will buy a new bike.
(-) If I do not pass the exam, I will not buy a new bike.
(?) Will you buy a new bike if you pass the exam?

## Conditional Sentence Type 2

> **Simply:** Unreal Present / Hypothetical.

**Formula:** If + Simple Past, Present Conditional (would + V1)

**Usage:** Commonly used for hypothetical or unreal situations in the present or future; often used for dreaming or imagining.

**Examples:**
(+) If I lived in Tokyo, I would go to that concert.
(-) If I did not live in Tokyo, I would not go to that concert.
(?) Would you go to that concert if you lived in Tokyo?

## Conditional Sentence Type 3

> **Simply:** Unreal Past / Regret.

**Formula:** If + Past Perfect, Perfect Conditional (would have + V3)

**Usage:** Commonly used for unreal past situations, often expressing regret about something that cannot be changed.

**Examples:**
(+) If I had left earlier, I would have caught the train.
(-) If I had not left earlier, I would not have caught the train.
(?) Would you have caught the train if you had left earlier?

## Mixed Conditional (Type 1: Past Action, Present Result)

> **Simply:** Past Cause -> Present Result.

**Formula:** If + Past Perfect, Present Conditional (would + V1)

**Usage:**  Commonly used to show how a past action affects the present situation. 

**Examples:**
(+) If I had studied harder, I would be successful now.
(-) If I had not studied harder, I would not be successful now.
(?) Would you be successful now if you had studied harder?

## Mixed Conditional (Type 2: Present Condition, Past Result)

> **Simply:** Present Condition -> Past Result.

**Formula:** If + Simple Past, Perfect Conditional (would have + V3)

**Usage:**  Commonly used to show how a permanent quality or present condition affected a past event.

**Examples:**
(+) If I were more organized, I would have finished the project yesterday.
(-) If I were not organized, I would not have finished the project yesterday.
(?) Would you have finished the project yesterday if you were more organized?
